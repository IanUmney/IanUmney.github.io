Level 5 Diploma in Web Application Development

It is a year-long level 5 diploma in the full-stack engineering of web applications.

It is self-paced and requires several submissions and objectives to achieve a passing mark.

The course is hosted by Newcastle College and is provided by CodeInstitute.


Course Structure

The course is divided into 4 units:

  • Unit 1: User Centric Frontend Development
  • Unit 2: Interactive Frontend Development
  • Unit 3: Data Centric Development
  • Unit 4: Full Stack Frameworks with Django
